# Club-Event-Management-Site
A responsive web application built for internal club management that allows authorized club members to create, edit, delete, and manage event details efficiently.
Developed using React.js for the frontend, Firebase Firestore for real-time database management, and Tailwind CSS for modern and responsive UI design.

🚀 Features

🔐 Member-only access for secure event management

➕ Add new events with detailed information

✏️ Edit existing event details

❌ Delete events

📅 View all upcoming and past events

⚡ Real-time database updates using Firestore

📱 Fully responsive UI with Tailwind CSS

🛠 Tech Stack

React.js

Firebase Firestore

Tailwind CSS
